What Adwa Auditor Actually Does
You upload a strategy (Python + data). We run it through forensic checks — lookahead detection, sandboxed execution, statistical vetoes — and give you a credibility score with a shareable PDF. Deployed on Oracle Cloud, verified via public portal.
🔍
IronDome AST Scanner
Static analysis that scans your strategy's Abstract Syntax Tree for lookahead bias — negative shifts, future indexing, hardcoded arrays. Instant DISQUALIFIED on violation.
🧪
Sandboxed Execution
Your strategy runs in a locked-down Python subprocess — 30s CPU, 512MB RAM, filesystem access blocked, dangerous modules (os, subprocess, requests) disabled. 3 runs with different seeds.
📊
Statistical Vetoes
Sharpe ratio > 8.0? 100% win rate? Zero volatility? Instant DISQUALIFIED. We catch strategies that look "too good to be true" — because they are.
⚖️
Sovereign Scale Scoring
Execution Realism, Causal Accuracy, and Reproducibility map to a 4-tier verdict: PASS, PASS WITH RISK, REVIEW, or REJECT. One credibility score to rule them all.
🔒
Cryptographic Ledger
Every audit gets a SHA-256 deterministic run hash. Public verification portal lets anyone check a result by hash. No tampering, no faking.
📄
PDF Report + QR Code
Pro+ audits get a cryptographic PDF with ASCII-scrubbed formatting and QR code linking to the verification portal. Share it with prop firms or keep it private.
